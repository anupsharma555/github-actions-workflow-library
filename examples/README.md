# Examples

This directory will hold small, public-safe examples for applying the workflow templates to different repo shapes.

Planned examples:

- Python app with backend tests and scripts.
- JavaScript or TypeScript frontend/app repo.
- Mixed Python plus JavaScript repo.
- Private-repo artifact-only mode.
- Public-repo GitHub Pages mode.

Each example should include:

- Files copied into `.github/`.
- Exclusion decisions.
- Expected outputs.
- Verification commands.
- Known caveats.
