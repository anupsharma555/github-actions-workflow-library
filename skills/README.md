# Codex Skill Alignment

This repository is designed to align with reusable Codex skills without embedding private skill contents into workflow templates.

Current related skills:

- `codex-general-repo-visualizer`: add, run, verify, and interpret GitHub OCTO Repo Visualizer workflows.
- `codex-general-emerge-github-workflow`: add, run, and verify Emerge GitHub workflows.
- `codex-general-emerge-analysis`: interpret Emerge metrics exports and save analysis reports.

Boundary:

- This public repository holds reusable workflow templates and public-safe docs.
- Local Codex skills hold operational guidance, parser scripts, and local workflow shortcuts.
- Repo-specific constraints belong in the target repo's docs, `AGENTS.md`, or local-only planning artifacts.
